A Website Style Guide

A style guide made for a sample website!

Colors

Blue

Blue

Blue is being used as a subheading background color with the help of: rgb(0, 38, 255)

Aquamarine

Aquarmarine

Aquamarine is being used as the heading color with the help of: hsl(160, 100%, 75%)

Blanched Almond

Blanched Almond

Blanched Almond is being used as the background color with the help of: #ffebcd


Fonts

Trirong

For the regular font of the page Trirong regular is being used

Example: The black dog jumps over the slow cat

Bold Trirong

For h2 headings Bold Trirong is being used

Example: The black dog jumps over the slow cat

Cursive

The website is using Cursive for Header h1 and h2 headings

Example: The black dog jumps over the slow cat


Text Style

Main heading

Main heading has a font size of 40px and is Bold

Section heading

Section headings has a font size of 25px and is Bold

Regular text

Text has a font size of 18px
